Preferences:

  • I prefer software that is local first. Clouds tend to have privacy issues, security issues, outages and anti-user incentives.
  • I prefer software written by small independent teams. Trusting some random person's personal repository is a lot easier for me to do than big tech corporations.
  • I prefer hosting my data in the EU where data privacy laws are typically better than in the US.
  • I prefer simple tools with fewer features as they tend to be more reliable, less disruptive and better at serving their niche.
  • I prefer to give services only the data they require about me to provide a service. I use sock puppet accounts when interacting with most services that do not need my identity.

If you have a better recommendation for one of these, let me know!

   

Servers & infrastructure

  • Synology
    • NAS
    • Plex
  • UniFi - NVR, Router, Firewall
  • Minisforum - Windows and Linux dedicated servers
  • DigitalOcean - virtual private servers, hosted block storage

   

Services

   

Mail

I exclusively use Proton Mail and I'm very happy with it.

   

Password Manager

  • I primarily use 1password. It has most of the features I want including passkeys and TOTP.
  • I also use ProtonPass because it allows me to create anonymous email addresses that forward to my inbox. I use these email addresses when signing up for new accounts.
  • I've heard great things about BitWarden and have no complains about Keeper which I use at work.
  • I've used LastPass in the past but migrated to 1Password because of better mobile app support at the time (many years ago). LastPass lost some trust after a significant data breach.

   

Documents

I begrudgingly use G's suite for now but intend to consider Synology Office at some point.

   

Payments

Privacy.com allows me to make purchases with temporary credit cards that do not need to be tied to my identity or location. This does require Privacy.com to have all of the information about me required to fulfill a purchase, but it does limit data exposure to other third parties.

   

Messaging

I use a handful of messaging applications. I have not found one I like enough to recommend (including Signal). Many of them require de-anonymizing steps such as requiring a phone number to activate or have very low adoption.

In the meantime my most preferred messaging application is Signal as it does have a number of good privacy and security features even if it requires additional data about users in order to function.

I intend to try hosting an XMPP server at some point but have no opinion on that yet.

   

Desktop

I use Manjaro Linux because it's very easy to use and I'm tired of configuring Linux for now. When I'm in the mood to tinker I like Void Linux.

I have machines running macOS and Windows 11 with minimal configuration - they are used for software testing and running services (such as building and testing my projects on those operating systems).

When installing Windows I've really liked using Christoph Schneegans' autounattend.xml generator to pre-configure many things about Windows prior to installation. An important feature of this tool (which is also possible to do manually) is the ability to create a local user account without creating an online Microsoft account.

   

Browser

I use LibreWolf with mostly default settings; the main exception is that I do clear history after closing the browser and manually add exceptions for sites that I want to stay logged in to. The privacy features that make tracking your activity across sessions and sites also change your timezone so that can't be used to uniquely identify you; it seems like a good idea but causes me too many issues in the usability of my browser so I work around that with the Spoof Timezone extension.

   

Programming

  • Sublime Text 3
  • nvim

Mobile

I use a Google Pixel 9a running GraphineOS. My plan is a pre-paid Verizon account using a fake name, fake address and a Privacy Card for payment.

   

App Store

  • App Store for applications designed for use with GraphineOS.
  • F-Droid for FOSS
  • Aurora Store to access the Google Play store anonymously.

   

Browser

  • DuckDuckGo Privacy Browser for normal browsing. I clear all data on app exit and fireproof only the sites I visit with accounts on them (Mastodon, GitHub, etc.)
  • Tor Browser for anonymous browsing.
